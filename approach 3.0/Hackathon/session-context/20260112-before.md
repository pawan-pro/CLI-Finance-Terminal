# Context Snapshot: Quantwater Terminal Build (Hackathon)
**Current Date:** January 12, 2026  
**Objective:** Build an institutional-grade CLI "AI Research Copilot" that analyzes proprietary market data exported from MetaTrader 5 (MT5) using Google Gemini 3.

## 1. Core Architecture (The Data Pipeline)
*   **Producer (MT5):** A hardened MQL5 Expert Advisor (`MarketSnapshot.mq5`) exports 15-minute (M15) bar data for 50+ symbols. It uses **Dynamic Lookback** (fetching the last 96 active bars) to automatically bridge weekends and holidays.
*   **Bridge (Symlink):** A macOS symbolic link connects the MT5 sandbox to the project’s `/data/bronze` folder for zero-latency data flow.
*   **Silver Layer (DuckDB):** A Python ingestion engine (`ingest_mt5_exports.py`) upserts raw CSVs into a high-performance **DuckDB** database with a `PRIMARY KEY (symbol, time_utc)` to prevent duplicates.
*   **Analytics Brain:** A SQL-based engine (`metrics_engine.py`) calculates 24h returns and ATR (volatility) relative to the latest available data point, handling timezone offsets (GMT+2 vs IST) automatically.
*   **AI Copilot:** A Gemini 3 integration (`research_copilot.py`) that receives structured JSON "Market Slabs" and synthesizes institutional research notes.
