## 1. Core Architecture (The Data Pipeline)
*   **Producers (Multi-Source):** 
    *   **MetaTrader 5 (MT5):** Exports M15 bar data for 50+ individual symbols. Features an aggressive sync loop to force-update broker caches.
    *   **Twelve Data (T12):** (Upgraded) Uses a **Rotational API Key Pool** (3+ keys) to bypass rate limits. Provides high-frequency (M15) data for **Bond Proxies** (SHY, IEF, TLT) and **All 11 S&P 500 Sectors** (XLK, XLF, XLE, etc.).
    *   **Synthetic Engine:** Calculates the **US Dollar Index ($DXY)** mathematically using a geometric weighted formula from the EUR, JPY, GBP, CAD, SEK, and CHF pairs in the lake.
*   **Bridge & Pathing:** Enforced **Forced Absolute Pathing** across all scripts to point to the master DuckDB file at `/Users/pawan/CLI-Finance-Terminal/approach 3.0/Hackathon/data/silver/market_data.duckdb`.
*   **Silver Layer (DuckDB):** 
    *   `fetch_high_freq.py`: Handles rotational T12 ingestion for 15 assets (Bonds + Sectors).
    *   `metrics_engine.py`: Features a **Price-to-Yield Transformer** to estimate Treasury yields and a sanitization layer for friendly names.
*   **Analytics Brain:** A SQL engine that now performs **Cross-Asset Calibration**. It converts ETF prices to yields and synthesizes the Dollar Index to provide a "Macro Gravity" context for the AI.
*   **AI Copilot:** A Gemini 1.5 Flash integration that now reasons using the **10Y/2Y Price Ratio** to detect regime shifts (e.g., "Reflationary Steepening").
