
# Context Snapshot: Quantwater Terminal Build (Hackathon)
**Current Date:** January 14, 2026  
**Objective:** Build an institutional-grade CLI "AI Research Copilot" that analyzes proprietary market data and economic events exported from MetaTrader 5 (MT5) using Google Gemini 3.

## 1. Core Architecture (The Data Pipeline)
*   **Producers (MT5):** 
    *   `MarketSnapshot.mq5`: (Hardened) M15 exporter for 50+ symbols. Features an **Aggressive Sync Loop** (3-strike retry) that handshakes with the broker to force-update stale caches for US Indices and Equities.
    *   `CalendarExport.mq5`: Extracts MT5 Economic Calendar data (Actual vs. Forecast) with automated Country Code mapping.
*   **Bridge (Symlink):** A macOS symbolic link connects the MT5 sandbox to the project’s `/data/bronze` folder for zero-latency data flow.
*   **Silver Layer (DuckDB):** 
    *   `ingest_mt5_exports.py`: Incremental ingestion of price CSVs into `m15_bars`. Now features a dedicated filter to exclude metadata/calendar files.
    *   `ingest_calendar.py`: Normalizes MT5 GMT+2 time to **UTC Standard** and upserts macro data into `economic_events` with duplicate-proof Primary Keys.
*   **Analytics Brain:** A SQL engine (`metrics_engine.py`) calculating **Verified 24h Rolling Returns**. Now includes **Institutional Sector Mapping** (isolating `METALS`) and a **Capital Flow Priority** logic to group assets by market importance.
*   **AI Copilot:** A Gemini 3 integration (`research_copilot.py`) featuring **Causal Reasoning**—mapping price spikes (e.g., Silver +6.53%) to specific high-impact economic "Surprises" (e.g., US CPI beat).

## 3. Current Outcome & Key Logic
*   **Sectoral Flow UI:** Developed the `sector` command which groups assets by institutional priority: **INDEX -> METALS -> FX -> CRYPTO -> STOCK**. Leveraged assets (`.lv`) are automatically reclassified from FX to CRYPTO.
*   **Temporal Precision:** Fully resolved the "Future Time" bug. Terminal IST now perfectly matches the local clock via a calibrated `+5.5h` UTC offset (or `+3.5h` from GMT+2 Broker time). 
*   **Auditability:** Every market quote in the CLI includes **Return Start (IST)** and **Last Sync (IST)**, providing a mathematically verifiable 24-hour observation window.
*   **Data Reliability:** Achieved a consistent 50/50 symbol sync success rate. The "Fighter" loop successfully extracts US500 and Mag7 equity bars that were previously sticking to old broker caches.
