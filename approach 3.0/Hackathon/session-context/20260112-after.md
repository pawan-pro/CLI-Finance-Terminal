# Context Snapshot: Quantwater Terminal Build (Hackathon)
**Current Date:** January 12, 2026  
**Objective:** Build an institutional-grade CLI "AI Research Copilot" that analyzes proprietary market data and economic events exported from MetaTrader 5 (MT5) using Google Gemini 3.

## 1. Core Architecture (The Data Pipeline)
*   **Producers (MT5):** 
    *   `MarketSnapshot.mq5`: Exports M15 bar data for 50+ symbols using **Dynamic Lookback** to bridge weekends.
    *   `CalendarExport.mq5`: (New) Extracts MT5 Economic Calendar data (Actual vs. Forecast) with automated Country Code mapping (e.g., 840 -> "US").
*   **Bridge (Symlink):** A macOS symbolic link connects the MT5 sandbox to the project’s `/data/bronze` folder for zero-latency data flow.
*   **Silver Layer (DuckDB):** 
    *   `ingest_mt5_exports.py`: Upserts price CSVs.
    *   `ingest_calendar.py`: (Deployed) Normalizes MT5 GMT+2 time to **UTC Standard** and upserts economic events into `economic_events` table with duplicate-proof Primary Keys.
*   **Analytics Brain:** A SQL engine (`metrics_engine.py`) calculating 24h returns, ATR, and **Volatility-Event Correlations** (joining price spikes with news releases).
*   **AI Copilot:** A Gemini 3 integration (`research_copilot.py`) synthesized with "Causal Awareness"—explaining price moves via "Surprise Deltas" in macro data.

## 3. Current Outcome & Key Logic
*   **UTC Standardization:** The entire lake now operates on UTC. Ingestion logic subtracts 2 hours from MT5 GMT+2 timestamps to ensure perfect alignment between price bars and news releases.
*   **Monday Morning Resilience:** The system now handles the "Weekend Gap" by allowing partial bar ingestion on Mondays, preventing the rejection of symbols while history buffers.
*   **Causal Awareness:** The terminal successfully exported 29 events for Jan 12, providing the "Why" (e.g., BoE Speech, Note Auctions) behind the "What" (Price Action).
*   **Deduplication:** Primary keys `(symbol, time_utc)` and `(time_utc, currency, event_name)` prevent database bloat from multiple daily runs.
