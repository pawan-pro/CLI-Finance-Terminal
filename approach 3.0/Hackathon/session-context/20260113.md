# Context Snapshot: Quantwater Terminal Build (Hackathon)
**Current Date:** January 13, 2026  
**Objective:** Build an institutional-grade CLI "AI Research Copilot" that analyzes proprietary market data and economic events exported from MetaTrader 5 (MT5) using Google Gemini 3.

## 1. Core Architecture (The Data Pipeline)
*   **Producers (MT5):** 
    *   `MarketSnapshot.mq5`: Verified 50-symbol M15 exporter. Uses **Dynamic Lookback** (fetching the last 96 active bars) to bridge weekends/holidays.
    *   `CalendarExport.mq5`: Extracts MT5 Economic Calendar data (Actual vs. Forecast) with automated Country Code mapping (e.g., 840 -> "US").
*   **Bridge (Symlink):** A macOS symbolic link connects the MT5 sandbox to the project’s `/data/bronze` folder for zero-latency data flow.
*   **Silver Layer (DuckDB):** 
    *   `ingest_mt5_exports.py`: Incremental ingestion of price CSVs into `m15_bars`.
    *   `ingest_calendar.py`: Normalizes MT5 GMT+2 time to **UTC Standard** and upserts macro data into `economic_events` with duplicate-proof Primary Keys.
*   **Analytics Brain:** A SQL engine (`metrics_engine.py`) calculating 24h returns, ATR, and **Temporal Joins** (mapping price moves to high-impact economic releases).
*   **AI Copilot:** A Gemini 3 integration (`research_copilot.py`) featuring **Causal Reasoning**—explaining price moves (e.g., Silver +11.79%) via anticipated macro triggers (e.g., US CPI expectations).
