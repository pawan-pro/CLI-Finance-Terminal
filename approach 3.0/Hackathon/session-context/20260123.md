# Context Snapshot: Quantwater Terminal Build (Hackathon)
**Current Date:** January 23, 2026  
**Objective:** Build an institutional-grade **Web-based AI Dashboard (Nexus Terminal)** that analyzes proprietary market data, bond yields, and sectoral flows using a FastAPI/DuckDB backend and Google Gemini 1.5 Flash.

## 1. Core Architecture (The Data Pipeline)
*   **Producers (Multi-Source):** 
    *   **MetaTrader 5 (MT5):** Exports M15 bar data for 50+ individual symbols. 
    *   **Twelve Data (T12):** Provides high-frequency (M15) history for **Bond Proxies** (SHY, IEF, TLT) and **All 11 S&P 500 Sectors**.
    *   **Synthetic Engine:** Calculates the **US Dollar Index ($DXY)** and **Treasury Yield Estimates** mathematically in the Silver Layer.
*   **Bridge Layer (FastAPI):** `app.py` acts as the master controller. It bridges the DuckDB Data Lake to the React Frontend, featuring a **15-minute server-side cache** for Gemini Research to prevent API quota exhaustion.
*   **Frontend (Nexus Terminal):** A dynamic React 19 / Tailwind UI. Features specialized views: **F1 (Equity Analysis)** and **F2 (Macro Dashboard)**.
*   **Symbol Handshake Logic:** Implemented a "Ticker-First" mapping in the backend SQL. Internal MT5 names (e.g., "Apple") are automatically served as standardized tickers (e.g., "AAPL") to ensure O(1) data binding in the UI.
